编写博客的语法

发表于 2020-05-21 Valine：
本文字数： 169 阅读时长 ≈ 1 分钟

编写博客的语法其实挺简单的

  • 标题

    在想要设置为标题的文字 一个#是一级标题，二个#是二级标题，以此类推。支持六级标题。注：标准语法一般在#后跟个空格再写文字，貌似简书不加空格也行。

    这是一级#

    这是二级##

    这是三级###

    这是四级####

    这是五级#####
    这是六级######

    -*+连续输入三个是分割线

    还有一些图片的格式，超链接格式，列表，嵌套等等。

博客的搭建

首先想搭建一个自己的博客，需要一下几种软件

  1. Hexo

  2. Node.js

  3. Git

  4. GitHub

    安装好上述软件就可以开始搭建属于自己的博客了（安装时一直点next，直到完成）

    • 安装Hexo 打开CMD输入npm install -g hexo出现版本号则成功。

    • 接着创建Hexo博客根目录 输入md blog>cd blog>hexo init 后出现Start blogging with Hexo!则本地搭建成功。

    • 那么恭喜你进入了博客的世界。

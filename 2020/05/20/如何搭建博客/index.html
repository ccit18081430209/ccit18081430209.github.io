博客的搭建

首先想搭建一个自己的博客，需要一下几种软件

  1. Hexo

  2. Node.js

  3. Git

  4. GitHub

    安装好上述软件就可以开始搭建属于自己的博客了（安装时一直点next，直到完成）

    • 安装Hexo 打开CMD输入npm install -g hexo出现版本号则成功。

    • 接着创建Hexo博客根目录 输入md blog>cd blog>hexo init 后出现Start blogging with Hexo!则本地搭建成功。

    • 那么恭喜你进入了博客的世界。
